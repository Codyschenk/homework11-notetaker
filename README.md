# cody-note-taker

## Description
This application is an online note taker to keep track of notes, save notes, and delete notes. You can add multiple notes, save them, and delete those that are no longer needed.

## User Story

```
AS A small business owner
I WANT to be able to write and save notes
SO THAT I can organize my thoughts and keep track of tasks I need to complete
```
